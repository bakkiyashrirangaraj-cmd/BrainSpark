## 7. Asset Delivery Formats

| Asset Type | Primary | Fallback | Notes |
|------------|---------|----------|-------|
| Icons | SVG | PNG @2x, @3x | Vector preferred |
| Mascots | Lottie | PNG sequence | Animation required |
| Badges | PNG | - | High detail |
| Sounds | MP3 | OGG | Web compatibility |
| Animations | Lottie | GIF | File size matters |
